# Proyecto: LePetitDep

## Descripción

LePetitDep es una aplicación web ficticia que permite a los usuarios solicitar café y productos de panadería para recogerlos en el local. La aplicación facilita la gestión de usuarios, exploración del menú, creación de pedidos y consulta de compras anteriores.

## Funcionalidades principales

- Registro de usuario.
- Inicio de sesión seguro.
- Exploración del menú de productos.
- Agregar productos al carrito de compra.
- Realizar pedidos con confirmación y método de pago simulado.
- Consulta del historial de pedidos.

## Historias de usuario

1. Registro de usuario.
2. Inicio de sesión.
3. Exploración del menú.
4. Agregar productos al carrito.
5. Realizar pedido.
6. Historial de pedidos.
