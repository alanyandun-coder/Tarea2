# Historia de Usuario 3: Exploración del menú

## Historia

Como cliente de LePetitDep,
quiero visualizar el menú disponible,
para conocer los productos, precios y características antes de realizar una compra.

## Criterios de aceptación

- El sistema debe mostrar la lista de productos disponibles.
- Cada producto debe incluir nombre, descripción y precio.
- Los productos deben estar organizados por categorías.
- El usuario debe poder consultar los detalles de cada producto.
- La información del menú debe estar actualizada.
