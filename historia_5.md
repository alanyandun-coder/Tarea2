# Historia de Usuario 5: Realizar pedido

## Historia

Como cliente,
quiero confirmar mi pedido y seleccionar un método de pago simulado,
para generar una orden y recoger mis productos en el local.

## Criterios de aceptación

- El usuario debe revisar el resumen del pedido antes de confirmarlo.
- El sistema debe permitir seleccionar un método de pago simulado.
- Al confirmar la compra, debe generarse un número de orden.
- El sistema debe mostrar la confirmación del pedido realizado.
- El pedido debe almacenarse en el historial del usuario.
