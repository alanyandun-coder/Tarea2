# Historia de Usuario 6: Historial de pedidos

## Historia

Como usuario registrado,
quiero consultar mis pedidos anteriores,
para revisar las compras realizadas anteriormente.

## Criterios de aceptación

- El usuario debe poder acceder a su historial de pedidos.
- El sistema debe mostrar fecha y número de orden.
- Cada pedido debe mostrar los productos adquiridos.
- El usuario debe poder consultar el estado de pedidos anteriores.
- La información mostrada debe corresponder únicamente al usuario autenticado.
