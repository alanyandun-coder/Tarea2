# Historia de Usuario 4: Agregar productos al carrito

## Historia

Como cliente de LePetitDep,
quiero agregar productos al carrito de compra,
para seleccionar los productos que deseo adquirir antes de realizar el pedido.

## Criterios de aceptación

- El usuario debe poder seleccionar productos del menú.
- El usuario debe elegir la cantidad de cada producto.
- El carrito debe mostrar los productos seleccionados.
- El sistema debe calcular el subtotal de la compra.
- El usuario debe poder modificar o eliminar productos del carrito.
