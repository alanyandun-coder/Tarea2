# Historia de Usuario 1: Registro de usuario

## Historia

Como nuevo cliente de LePetitDep,
quiero crear una cuenta utilizando mi correo electrónico y contraseña,
para poder acceder a los servicios de la aplicación y realizar pedidos.

## Criterios de aceptación

- El usuario debe ingresar un correo electrónico válido.
- El usuario debe ingresar una contraseña segura.
- El sistema debe validar que el correo no esté registrado previamente.
- Al completar el registro correctamente, el sistema debe mostrar un mensaje de confirmación.
- Los datos del usuario deben almacenarse de manera segura.
